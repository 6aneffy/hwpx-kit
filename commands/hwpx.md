아래 카탈로그를 그대로 사용자에게 출력하라. 다른 도구를 호출하지 말 것.
(이 카탈로그는 사용자가 /hwpx를 직접 쳤을 때만 보여주는 것 — 문서 작업
요청이 들어온 상황이라면 카탈로그 출력 없이 해당 스킬로 바로 진행하라.)

```
📄 hwpx-kit — 한글(HWPX) 문서 자동화

  스킬                이렇게 말하면 됩니다
  ─────────────────────────────────────────────────────────────
  hwpx-form          "이 양식에 채워줘" (hwpx/hwp 파일과 함께)
                     "이 문서 읽어줘 / 요약해줘"
                     "이 기획서를 ○○사업용으로 바꿔줘"
  doc-create         "한글로 새 기획서 만들어줘" (파일 없이)
  format-convert     "금액 한글로 병기해줘" / "날짜에 요일 붙여줘"
  gongmun-format     "공문 형식으로 정리해줘"
  table-calc         "표에 증감·비율·합계 채워줘"
  office-export      "이거 워드로 줘" / "PPT로 만들어줘"

  원칙: 원본은 절대 수정하지 않고 사본에만 씁니다.
        서식(폰트·표·조판)은 그대로 보존됩니다.
        필수 정보는 먼저 여쭤보고, 지어내지 않습니다.

  요구사항: Python 3.10+ (첫 사용 때 CLI 자동 설치 안내)
  워드 변환·hwp 변환은 Windows + 한글(한컴오피스) 필요

  문서·이슈: https://github.com/6aneffy/hwpx-kit
```

출력 후, 사용자가 하려는 작업이 보이면 해당 스킬로 자연스럽게 이어가라.
